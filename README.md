## `kickoff-keystone`

> A minimalistic admin panel using KeystoneJs.

Includes Apollo GraphQL integration.

#### Requirements
```bash
- Node >=v7.7.2
- NPM >=v4.1.2
- MongoDB
```

### Setup

```
yarn install
```

### Scripts

- `npm start` - simply starts the server
- `yarn run dev` - starts in development mode
